## RowLocation

> **RowLocation** = `number` \| [`RowLocationWithAlign`](../scroll-location/#rowlocationwithalign)

A location in the list to scroll to. Passing a number scrolls instantly to the row at the specified index aligned to the top. See [RowLocationWithAlign](../scroll-location/#rowlocationwithalign) for more advanced options.


---

## RowLocationWithAlign

Specifies a location in the list to scroll to.

### Properties

#### align?: `"start"` \| `"center"` \| `"end"` \| `"start-no-overflow"`


How to align the item in the viewport.

***

#### behavior?: [`ScrollBehavior`](../scroll-animation/#scrollbehavior)


Set `'smooth'` to have an animated transition to the specified location.

***

#### done()?

> `optional` **done**: () => `void`

A callback that's invoked when the scroll is complete.

##### Returns

`void`

***

#### index

`number` \| `"LAST"`

The index of the item to scroll to. Use `'LAST'` to scroll to the last item.

***

#### offset?: `number`


Use the offset for additional adjustment of the position - can be a positive or negative number.
