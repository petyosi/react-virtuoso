## useCurrentlyRenderedData

> **useCurrentlyRenderedData**\<`Data`\>(): `Data`[]

Lets you access the currently rendered data items.

### Type Parameters

| Type Parameter | Description |
| ------ | ------ |
| `Data` | The type of the data items in the table. |

### Returns

`Data`[]

## useVirtuosoLocation

> **useVirtuosoLocation**(): [`ListScrollLocation`](../state/#listscrolllocation)

Lets you access the current scroll location of the data table component from its child components. See [ListScrollLocation](../state/#listscrolllocation) for the available properties.

[`ListScrollLocation`](../state/#listscrolllocation)

## useVirtuosoMethods

> **useVirtuosoMethods**\<`Data`\>(): [`VirtuosoDataTableMethods`](../imperative-api/#virtuosodatatablemethods)\<`Data`\>

### Type Parameters

| Type Parameter | Default type | Description |
| ------ | ------ | ------ |
| `Data` | `unknown` | The type of the data items in the table. |

[`VirtuosoDataTableMethods`](../imperative-api/#virtuosodatatablemethods)\<`Data`\>
