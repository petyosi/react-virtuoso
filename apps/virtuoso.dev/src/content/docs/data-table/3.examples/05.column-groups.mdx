Column groups are ideal when several fields form one mental unit, such as inventory metrics or financial metrics.

## APIs that matter

- `ColumnGroup`
- `ColumnGroupHeader`

```tsx live wide file=App.tsx
import { ColumnGroup, ColumnGroupHeader } from '@/components/ui/data-table'
import { DataTable, DataTableCell, DataTableColumn, DataTableColumnHeader } from '@/components/ui/data-table'

import { products } from './data'

const currency = new Intl.NumberFormat('en-US', {
  currency: 'USD',
  style: 'currency',
})

export default function App() {
  return (
    <DataTable className="rounded-xl" data={{ data: products, groups: [] }} style={{ height: 400 }}>
      <DataTableColumn field="name" sticky="left">
        <DataTableColumnHeader>Product</DataTableColumnHeader>
        <DataTableCell className="font-medium">{({ row }) => row.data.name}</DataTableCell>
      </DataTableColumn>

      <ColumnGroup>
        <ColumnGroupHeader className="px-2 text-center text-xs font-semibold uppercase tracking-wide text-muted-foreground">
          {() => 'Inventory'}
        </ColumnGroupHeader>
        <DataTableColumn field="stock">
          <DataTableColumnHeader className="justify-end">Stock</DataTableColumnHeader>
          <DataTableCell className="text-right">{({ cellValue }) => String(cellValue)}</DataTableCell>
        </DataTableColumn>
        <DataTableColumn field="status">
          <DataTableColumnHeader>Status</DataTableColumnHeader>
          <DataTableCell>{({ cellValue }) => String(cellValue)}</DataTableCell>
        </DataTableColumn>
      </ColumnGroup>

      <ColumnGroup>
        <ColumnGroupHeader className="px-2 text-center text-xs font-semibold uppercase tracking-wide text-muted-foreground">
          {() => 'Commercial'}
        </ColumnGroupHeader>
        <DataTableColumn field="price">
          <DataTableColumnHeader className="justify-end">Price</DataTableColumnHeader>
          <DataTableCell className="text-right tabular-nums">{({ cellValue }) => currency.format(Number(cellValue))}</DataTableCell>
        </DataTableColumn>
        <DataTableColumn field="revenue">
          <DataTableColumnHeader className="justify-end">Revenue</DataTableColumnHeader>
          <DataTableCell className="text-right tabular-nums">{({ cellValue }) => currency.format(Number(cellValue))}</DataTableCell>
        </DataTableColumn>
      </ColumnGroup>
    </DataTable>
  )
}
```

```ts live file=data.ts
export const products = Array.from({ length: 100 }, (_, index) => {
  const stock = (index * 5) % 32

  return {
    id: `SKU-${String(index + 1).padStart(3, '0')}`,
    name: `Product ${index + 1}`,
    stock,
    status: stock < 6 ? 'Backorder' : stock < 12 ? 'Low stock' : 'In stock',
    price: 49 + (index % 11) * 13,
    revenue: 1_100 + index * 85,
  }
})
```

## Tradeoffs

Grouped headers improve scannability, but they require more vertical header space and clearer column naming.
