Column groups are a header structure feature. Wrap related `DataTableColumn` declarations in
`ColumnGroup` and provide a `ColumnGroupHeader`.

```tsx
<ColumnGroup>
  <ColumnGroupHeader>{() => 'Inventory'}</ColumnGroupHeader>
  <DataTableColumn field="stock">
    <DataTableColumnHeader>Stock</DataTableColumnHeader>
    <DataTableCell>{({ cellValue }) => String(cellValue)}</DataTableCell>
  </DataTableColumn>
  <DataTableColumn field="status">
    <DataTableColumnHeader>Status</DataTableColumnHeader>
    <DataTableCell>{({ cellValue }) => String(cellValue)}</DataTableCell>
  </DataTableColumn>
</ColumnGroup>
```

Column groups can be combined with sticky columns and column reordering. For group-level drag handles,
use `DraggableGroupHeader` from the shadcn column-reorder registry component.

See the [Column Groups example](/data-table/examples/column-groups/) for a complete grouped-header
layout with sticky columns.
