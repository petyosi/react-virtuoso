`VirtuosoDataTable` measures what your headers and cells render. That means layout is driven by real content, not by a separate column-width config unless you choose to override it later.

Two practical rules help immediately:

- use consistent padding and alignment in every column
- let the rightmost sticky column react to `overlaidByScrollbar` so macOS overlay scrollbars do not cover content

## Styled financial view

```tsx live wide file=App.tsx
import { DataTable, DataTableCell, DataTableColumn, DataTableColumnHeader } from '@/components/ui/data-table'

import { products } from './data'

const currency = new Intl.NumberFormat('en-US', {
  currency: 'USD',
  style: 'currency',
})

export default function App() {
  return (
    <DataTable className="rounded-xl" data={{ data: products, groups: [] }} style={{ height: 400 }}>
      <DataTableColumn field="name" sticky="left">
        <DataTableColumnHeader>Product</DataTableColumnHeader>
        <DataTableCell>
          {({ row }) => (
            <div className="flex min-w-[220px] flex-col">
              <span className="font-medium">{row.data.name}</span>
              <span className="text-xs text-muted-foreground">{row.data.vendor}</span>
            </div>
          )}
        </DataTableCell>
      </DataTableColumn>

      <DataTableColumn field="category">
        <DataTableColumnHeader>Category</DataTableColumnHeader>
        <DataTableCell>{({ cellValue }) => String(cellValue)}</DataTableCell>
      </DataTableColumn>

      <DataTableColumn field="price">
        <DataTableColumnHeader className="min-w-[120px] justify-end">Price</DataTableColumnHeader>
        <DataTableCell className="min-w-[120px] text-right tabular-nums">
          {({ cellValue }) => currency.format(Number(cellValue))}
        </DataTableCell>
      </DataTableColumn>

      <DataTableColumn field="revenue">
        <DataTableColumnHeader className="min-w-[140px] justify-end">Revenue</DataTableColumnHeader>
        <DataTableCell className="min-w-[140px] text-right tabular-nums">
          {({ cellValue }) => currency.format(Number(cellValue))}
        </DataTableCell>
      </DataTableColumn>

      <DataTableColumn field="status" sticky="right">
        <DataTableColumnHeader>
          {({ overlaidByScrollbar }) => (
            <div
              className="w-full min-w-[140px] text-right"
              style={{
                paddingRight: overlaidByScrollbar ? 'calc(0.75rem + var(--overlay-scrollbar-visible-size))' : '0.75rem',
              }}
            >
              Status
            </div>
          )}
        </DataTableColumnHeader>
        <DataTableCell>
          {({ row, overlaidByScrollbar }) => (
            <div
              className="w-full min-w-[140px] text-right"
              style={{
                paddingRight: overlaidByScrollbar ? 'calc(0.75rem + var(--overlay-scrollbar-visible-size))' : '0.75rem',
              }}
            >
              <span
                className={
                  row.data.status === 'Backorder'
                    ? 'font-medium text-red-600'
                    : row.data.status === 'Low stock'
                      ? 'font-medium text-amber-600'
                      : 'font-medium text-emerald-600'
                }
              >
                {row.data.status}
              </span>
            </div>
          )}
        </DataTableCell>
      </DataTableColumn>
    </DataTable>
  )
}
```

```ts live file=data.ts
export interface Product {
  id: string
  name: string
  category: string
  price: number
  revenue: number
  status: 'In stock' | 'Low stock' | 'Backorder'
  vendor: string
}

const categories = ['Office', 'Peripherals', 'Audio', 'Software']
const vendors = ['Northwind', 'Acme', 'Orbit', 'Signal']

export const products: Product[] = Array.from({ length: 140 }, (_, index) => {
  const stock = (index * 7) % 48

  return {
    id: `SKU-${String(index + 1).padStart(3, '0')}`,
    name: `${categories[index % categories.length]} Item ${index + 1}`,
    category: categories[index % categories.length]!,
    price: 49 + (index % 11) * 13,
    revenue: 1_400 + index * 97,
    status: stock < 8 ? 'Backorder' : stock < 16 ? 'Low stock' : 'In stock',
    vendor: vendors[index % vendors.length]!,
  }
})
```

## Measurement tips

- Keep header and cell content widths aligned. A header with a tiny label and a cell with a long layout causes unnecessary width jumps.
- Give numeric columns a minimum width when the formatting can grow.
- Treat the overlay scrollbar case as part of the default design if you use a sticky right column.
