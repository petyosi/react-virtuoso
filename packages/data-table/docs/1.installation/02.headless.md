Choose the headless package when your team already has a design system, uses non-Tailwind styling, or wants to build a table wrapper with fully custom markup.

## Install the package

```bash
pnpm add @virtuoso.dev/data-table
```

You can use the same package with `npm`, `yarn`, or `bun` if that matches your workspace.

## Import structural styles

The package ships a small CSS file with layout defaults (column sizing, box model) inside a `@layer virtuoso` block. Import it once in your app's CSS or entry point:

```css
@import '@virtuoso.dev/data-table/styles.css';
```

Or in a JS/TS entry:

```typescript
import '@virtuoso.dev/data-table/styles.css'
```

These defaults have the lowest cascade priority -- any plain CSS or Tailwind utility class overrides them automatically.

## Minimal headless example

```tsx live wide file=App.tsx
import { Cell, Column, ColumnHeader, VirtuosoDataTable, localModel } from '@virtuoso.dev/data-table'

import { rows } from './data'

const currency = new Intl.NumberFormat('en-US', {
  currency: 'USD',
  style: 'currency',
})

const model = localModel({ data: rows })

export default function App() {
  return (
    <VirtuosoDataTable
      model={model}
      style={{
        height: 360,
        border: '1px solid var(--border)',
        borderRadius: 16,
        overflow: 'hidden',
      }}
    >
      <Column field="name">
        <ColumnHeader>
          {() => <div style={{ padding: '0 12px', fontWeight: 600, height: 44, display: 'flex', alignItems: 'center' }}>Product</div>}
        </ColumnHeader>
        <Cell>
          {({ cellValue }) => (
            <div style={{ padding: '0 12px', height: 44, display: 'flex', alignItems: 'center' }}>{String(cellValue)}</div>
          )}
        </Cell>
      </Column>
      <Column field="category">
        <ColumnHeader>
          {() => <div style={{ padding: '0 12px', fontWeight: 600, height: 44, display: 'flex', alignItems: 'center' }}>Category</div>}
        </ColumnHeader>
        <Cell>
          {({ cellValue }) => (
            <div style={{ padding: '0 12px', height: 44, display: 'flex', alignItems: 'center' }}>{String(cellValue)}</div>
          )}
        </Cell>
      </Column>
      <Column field="price">
        <ColumnHeader>
          {() => (
            <div
              style={{ padding: '0 12px', fontWeight: 600, height: 44, display: 'flex', alignItems: 'center', justifyContent: 'flex-end' }}
            >
              Price
            </div>
          )}
        </ColumnHeader>
        <Cell>
          {({ cellValue }) => (
            <div
              style={{
                padding: '0 12px',
                height: 44,
                display: 'flex',
                alignItems: 'center',
                justifyContent: 'flex-end',
                fontVariantNumeric: 'tabular-nums',
              }}
            >
              {currency.format(Number(cellValue))}
            </div>
          )}
        </Cell>
      </Column>
    </VirtuosoDataTable>
  )
}
```

## When headless is the better choice

- You need CSS-in-JS, CSS modules, or a non-Tailwind stack.
- You want to integrate with an existing internal component library.
- You need custom row wrappers, scroll elements, or column renderers that do not fit the default wrapper.

The headless package gives you the full surface area, including local and remote data models, table
instance control, scroll control, and testing utilities.

After installation, continue with [data model](/data-table/data-model/) and
[columns](/data-table/columns/).
