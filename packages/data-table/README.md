# React Virtuoso Data Table

The `VirtuosoDataTable` is a virtualized React data table component with both column and row virtualization. In addition to virtualized rendering out of the box, the component exposes an imperative data management API that gives you the necessary control over the scroll position when data changes. The scroll position can change instantly or with a smooth scroll animation.

To learn more, visit [the website](https://virtuoso.dev/data-table/).
