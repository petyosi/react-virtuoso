# Design work

The old site has outdated branding assets that need refreshing. The new site already has a basic favicon.svg but needs a complete set of modern favicon formats, a professionally designed social card for link previews, and improved navigation styling.

## Asset Requirements & Configuration

### Starlight Asset Configuration

- **Favicon**: Set via `favicon` config option (defaults to `/favicon.svg`), supports .ico, .gif, .jpg, .png, .svg
- **Logo**: Set via `logo` config option with `src` or separate `light`/`dark` variants for theme support
- **Social Cards**: No built-in support - requires custom `head` meta tags globally or per-page via route middleware
- **Multiple favicon sizes**: Added via `head` config for platform-specific icons

References:

- [Starlight Configuration](https://starlight.astro.build/reference/configuration/)
- [Add Open Graph images to Starlight](https://hideoo.dev/notes/starlight-og-images/)
- [Favicon configuration discussion](https://github.com/withastro/starlight/discussions/1058)
